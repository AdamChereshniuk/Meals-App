## Проект: Meals App 🍲

**Meals App** — это react-приложение для поиска блюд по названию, первой букве и ингредиентам.


🌐 Просмотр:
[Meals App](https://adamchereshniuk.github.io/Meals-App/)

### 🚀 Установка:

Установка зависимостей:
```
npm install
```
Запуск сервера разработки:
```
npm run dev
```
Запуск production сборки:
```
npm run build
```

### 🛠️ Технологии:

- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
